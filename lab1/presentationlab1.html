Установка Linux в VirtualBox

Цель работы

  • Изучить процесс виртуализации
  • Научиться создавать виртуальные машины
  • Установить ОС Linux (Fedora) в VirtualBox

Используемое ПО

  • Операционная система Linux
  • VirtualBox
  • ISO-образ Fedora

Подготовка среды

  • Создание каталога для ВМ
  • Настройка папки виртуальных машин
  • Проверка параметров VirtualBox

Настройка хост-клавиши

  • Хост-клавиша по умолчанию — Right Ctrl
  • При необходимости изменена
  • Исключён конфликт с переключением языка

Создание виртуальной машины

  • Тип ОС: Fedora (64-bit)
  • ОЗУ: 2048 МБ
  • Диск: VDI, 80 ГБ
  • Контроллер: IDE

Подключение ISO-образа

  • Добавлен виртуальный DVD-привод
  • Подключён ISO-образ Fedora
  • Установлен приоритет загрузки с DVD

Дополнительные настройки

  • Графический контроллер VMSVGA
  • Включено 3D-ускорение
  • Буфер обмена — двусторонний
  • Поддержка UEFI

Установка Fedora

  • Выбор языка
  • Разметка диска
  • Создание пользователя
  • Установка системы

Результат работы

  • Fedora успешно установлена
  • Виртуальная машина готова к работе
  • Система корректно загружается

Вывод
