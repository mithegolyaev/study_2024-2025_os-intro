# Что такое Git?

-   Распределённая система контроля версий\
-   Хранит историю изменений\
-   Позволяет работать в ветках\
-   Поддерживает подпись коммитов

------------------------------------------------------------------------

# Установка ПО (Fedora)

``` bash
sudo dnf install git
sudo dnf install gh
```

Проверка версии:

``` bash
git --version
gh --version
```

------------------------------------------------------------------------

# Базовая настройка Git

``` bash
git config --global user.name "Name Surname"
git config --global user.email "work@mail"
git config --global core.quotepath false
git config --global init.defaultBranch master
git config --global core.autocrlf input
git config --global core.safecrlf warn
```

------------------------------------------------------------------------

# Создание SSH-ключей

RSA 4096 бит:

``` bash
ssh-keygen -t rsa -b 4096
```

Ed25519:

``` bash
ssh-keygen -t ed25519
```

Добавление публичного ключа в GitHub.

------------------------------------------------------------------------

# Создание PGP-ключа

``` bash
gpg --full-generate-key
```

Параметры: - Тип: RSA and RSA\
- Размер: 4096\
- Срок действия: без ограничения

------------------------------------------------------------------------

# Подписание коммитов

``` bash
git config --global user.signingkey <PGP_FINGERPRINT>
git config --global commit.gpgsign true
git commit -S -m "Initial commit"
```

Коммиты будут отмечены как Verified на GitHub.
